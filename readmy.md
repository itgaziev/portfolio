# Portfolio: PHP/Bitrix Backend Solutions

В данном разделе представлены ключевые модули и интеграционные решения, разработанные мною. Код демонстрирует навыки проектирования отказоустойчивых систем, работы с внешними API и автоматизации e-commerce процессов.

---

### 📦 Интеграция с маркетплейсами и учетными системами

#### 1. **itgaziev.synch — Ядро синхронизации (Ozon, WB, Yandex)**
Модуль для двустороннего обмена данными между Bitrix и кастомным расширением 1С «Мастер заказов».
* **Функционал:** Синхронизация статусов заказов, остатков и цен в реальном времени.
* **Технологии:** REST API, интеграция с кастомными объектами 1С.
* **Результат:** Единое окно управления заказами из трех маркетплейсов внутри учетной системы.

#### 2. **itgaziev.ozonyml — Генератор фидов**
Выгрузка товарного каталога в формате YML, оптимизированная под требования Ozon.
* **Особенности:** Поддержка многопоточности при генерации больших файлов, гибкая настройка соответствия категорий.
* **Стек:** XML/YML, Bitrix ORM.

#### 3. **itcmedia.yaimport — Интеллектуальный импорт (Yandex Disk)**
Автоматизация наполнения каталога данными из облачного хранилища.
* **Задача:** Загрузка Excel-прайсов и автоматический поиск изображений товаров по артикулу на Яндекс.Диске.
* **Решение:** Реализован пошаговый механизм обработки (stepper) для обхода лимитов PHP по времени выполнения.
* **Стек:** Yandex Disk API, PHP Spreadsheet.

### 📊 Автоматизация и отчетность

#### 6. **itcmedia.report — Аналитическая панель продаж**
Модуль для формирования управленческой отчетности в админ-панели Bitrix.
* **Сложность:** Консолидация данных о продажах с множества разрозненных складов и учет себестоимости.
* **UI/UX:** Интерактивные отчеты с фильтрацией по периодам и категориям.

#### 7. **itgaziev.pricecalc — Динамическое ценообразование**
Инструмент для автоматического управления маржинальностью.
* **Функционал:** Автоматическая установка наценки на группы товаров на основе заданных фильтров и типов цен. Позволяет массово изменять стратегию продаж в несколько кликов.
